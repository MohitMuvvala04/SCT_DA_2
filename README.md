# SCT_DA_2 - Data Cleaning and Preparation

## Objective

The objective of this task is to clean and prepare a dataset for analysis using Python and Pandas.

## Dataset

Sample Superstore Dataset

## Tools Used

* Python
* Pandas
* Google Colab

## Steps Performed

1. Loaded the dataset into Pandas.
2. Inspected the dataset using `df.info()`.
3. Checked for missing values using `df.isnull().sum()`.
4. Checked for duplicate records using `df.duplicated().sum()`.
5. Converted `Order Date` and `Ship Date` columns to datetime format.
6. Exported the cleaned dataset as a CSV file.

## Results

* Missing Values: 0
* Duplicate Records: 0
* Date columns successfully converted to datetime format.
* Cleaned dataset exported successfully.

Successfully cleaned and prepared the dataset for further analysis and visualization.
